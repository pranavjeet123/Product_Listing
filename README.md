# Product Listing

Scaffolded React 19 application using Vite.

Quick start

1. Install dependencies:

```bash
npm install
```

2. Run dev server:

```bash
npm run dev
```

Open the URL shown by Vite (usually http://localhost:5173).

Notes

- Uses `react` and `react-dom` ^19.0.0 and Vite for fast dev feedback.
- Adjust or add components under `src/` as needed.
